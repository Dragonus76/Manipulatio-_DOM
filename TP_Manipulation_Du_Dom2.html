Languages basés sur EMCAScript

Languages basés sur ECMASScript :

JavaScript
JavaScript est un languages de programmation de scripts principalement utilisé dans les pages web interactives mais aussi coté serveur .
JScript
Jscript est le nom générique de plusieurs implémentations d'ECMAScript 3 crées par Microsoft.
ActionScript
ActionScript est le langage de programmation utilisé au sein d'applications clientes(Abode Flash, Abode Flex) et serveur (Flash media serer, Jrun, Macromedia Generator).
EX4
EMCMAScript for XML (E4X) est une extention XML au langage ECMAScript.
